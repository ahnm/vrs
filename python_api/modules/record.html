pyvrs.record module¶

class pyvrs.record.VRSBlocks(get_func: Callable[[int], T], range_: range)[source]¶

Bases: Sequence

A representation of a list of Record blocks (image, metadata, audio or custom). Accessing elements returns the type directly.
